# 주문관리 API 서비스 개발 (only Backend)

### 기술스텍

- java 8
- springBoot 2.x
- JPA
- gradle
- Junit 5

<br>

### 운영방안

- 간단한 디자인으로 만들어진 front에 RestAPI를 만든다.
- 평일 하루 2시간 이상 작업할 것

### 서비스 개발 아이디어 노트

어떤 식으로 서비스를 만들지에 대한 아이디어를 생각 날 때마다 기록한다.
